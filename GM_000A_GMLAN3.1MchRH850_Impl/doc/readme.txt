Intro
=======================================================================
This document describes the installation of a GMLAN delivery package.


=======================================================================
Installation
=======================================================================
The package will be installed by executing the file setup.exe.

It will install the GMLAN handler files, the documentation and sample 
files for a certain micro.

Please follow the installation instructions during the installing 
process.  

If you need any help or you have some questions please contact one
of our representatives listed below.
